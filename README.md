## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = 32px></h2>
This app is a competition and shows how developers can use the class structure. The class has two instance variables and one constructor. This constructor only accepts the required named parameter. As you can see, constant variables are used in this project. That's why we put the letter 'k' at the beginning of the variables. In addition, a list type variable and a variable for question indexes were used in the competition questions class. These variables are set to private. Thanks to this, No one can't direct access to them. On the other hand, a function was used for buttons. Developers can see the usage AlertDialog Widget in this function. Thanks to this widget, we create an information screen at the end of the competition. Lastly, we were used to Wrap Widget for checking our answers.
